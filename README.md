# Displine-Management-System
This a window form application made in java using jdbc api and the Data access Object
The project is here to implement the full features needed by the Secondary school Displine in Charges
Like: 
Granting student permission;
Recording student permission;
Keeping student information;
Sending sms and email to guardians to parents when the student records are changed at school
generating pdfs for permission and punishemnts;
the system also has a user authentication system and Acces control
